## 🚀 Quick Start

### 1. Cài đặt Dependencies

```bash
pip install -r requirements.txt
```

### 2. Cấu hình Environment

```bash
# Copy template và điền thông tin
cp config/env_template.txt .env
# Edit .env file với thông tin thực tế
```

### 3. Chạy Demo System

```bash
python scripts/demo_integrated_system.py
```

### 4. Chạy Full System

```bash
python scripts/start_system.py
```

## 🔧 Cấu hình

### System Configuration
Chỉnh sửa file `config/system_config.yaml` để cấu hình hệ thống:

```yaml
trading:
  symbol: "XAUUSDc"
  timeframe: 1
  risk_per_trade: 0.02
  
ai_ml:
  learning_rate: 0.001
  batch_size: 64
  
performance_targets:
  win_rate: 0.85
  sharpe_ratio: 3.5
```

### Environment Variables
Tạo file `.env` từ template `config/env_template.txt` và điền:
- MT5 credentials
- API keys
- Database URLs
- Notification settings
